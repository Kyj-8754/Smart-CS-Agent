# 스마트 CS 에이전트 프론트엔드

이 디렉토리는 [React](https://react.dev/)와 [Vite](https://vitejs.dev/)를 사용하여 구축된 웹 애플리케이션입니다.
백엔드 API와 통신하여 채팅 인터페이스 및 트랜잭션 승인 UI를 제공합니다.

## 개발 서버 실행

개발 모드에서 프론트엔드를 실행하려면 다음 명령어를 사용하세요:

```bash
npm install
npm run dev
```

## 주요 기능 (예정)

- **채팅 인터페이스**: AI 상담원과 실시간 대화.
- **승인 UI**: 주문 취소/정보 변경 등 중요 작업에 대한 사용자 승인 버튼 제공.
- **반응형 디자인**: 다양한 기기에서 최적화된 화면 제공.

## 기술 스택

- **Vite**: 빠른 빌드 및 HMR 지원.
- **React**: UI 컴포넌트 라이브러리.
- **CSS**: 커스텀 스타일링 (Premium Design 적용).
